# Industrial PLC Conveyor Sorting Simulator
# Overview

This project simulates an industrial conveyor sorting system using PLC concepts implemented in Python.

The system demonstrates how a Programmable Logic Controller controls conveyor movement, detects products using sensors, makes sorting decisions, and safely controls actuators through ladder logic.

The simulator also includes a Human Machine Interface (HMI) that visualizes the complete sorting process.
